# Project Gorgon Multitool

Project Gorgon Multitool is a Windows desktop utility designed to support your gameplay sessions with practical, readable tools that stay useful while you play.

The goal is simple: keep whispers, combat data, fight history, chat, and maps easy to follow without having to manage everything inside the main game window.

## What it does

### Whisper Overlay
Shows incoming whispers directly in an overlay, so you do not miss important messages while fighting, traveling, or focusing on gathering.

### DPS Overlay
Gives you an immediate view of the current fight, including:
- current DPS
- total damage
- player damage
- pet damage
- current target
- optional damage taken

It is designed to stay easy to read at a glance without taking over the screen.

### Fight History
Keeps track of completed fights and lets you quickly review:
- duration
- total damage
- player and pet contribution
- last target
- average DPS

### Fight Details
For each recorded fight, you can open a detailed breakdown showing:
- damage sources
- skills used
- recent hits
- fight target
- average DPS for the encounter

### External Chat
Opens a separate chat window from the game, making messages easier to read without mixing everything into the main interface.

It also includes useful features such as:
- custom channel colors
- hiding channels you do not want to see
- keyword highlighting
- optional sound alerts

### Egg Run Tools
Includes dedicated Egg Run tools with a control window and overlay support.

Currently supported servers:
- Dreva
- Arisetsu
- Strekios

The selected server is saved between sessions, and the active server name is shown in both the control window and overlay.

### Maps
Includes a built-in map browser and map editor.

You can:
- browse available maps
- open a full map editor
- pan and zoom smoothly
- draw with pen and eraser tools
- place text using a drag-to-create text box workflow
- change annotation color
- undo the last change
- clear all annotations

If a map has already been edited, the browser preview shows the user-modified version instead of the clean base map.

Map source credit: [Project Gorgon Wiki](https://wiki.projectgorgon.com/)

### Session Utility
From the main window you can quickly manage your session:
- start and stop log watching
- reset session data
- enable or disable overlays
- open or close the external chat window
- open Egg Run tools
- open Maps

## Why use it

Project Gorgon has a lot of useful information in its logs and chat, but during gameplay it is not always convenient to keep track of everything.

This tool is built to make the most useful information easier to follow:
- avoid missing important whispers
- see how a fight is going at a glance
- review damage, pet contribution, and target information quickly
- separate useful chat from noise
- review the details of a fight right after it ends
- annotate maps for your own routes, notes, and references
- keep a cleaner and more focused play experience

## Quick Start

1. Make sure your Project Gorgon account has **VIP** access.
2. In Project Gorgon, enable **chat log** output.
3. Download the latest release.
4. Launch the tool.
5. Press **Start**.
6. Enable the overlays you want to use.
7. Open External Chat, Egg Run tools, or Maps as needed.

Once started, the tool begins showing useful information through its windows and overlays during your session.
